wtyczka Usage Instructions

wtyczka add-in can be used to connect loose sketch points automatically or manually


Steps:

>1. Choose the connection type from the dropdown list "Join Type". Available options are "Join", "Fill", and "Table Select".

2. For "Table Select" mode, a table will appear showing all potential connections - check the ones you want to make.

3. For "Join" or "Fill" modes, optionally check "Manual Accept Connections" if you want to manually approve each connection.

4. If you need assistance, click the "Help" button.

Check if the sketch connections have been made.
